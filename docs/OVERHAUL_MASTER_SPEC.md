# VAMPIRE CITY — Master Overhaul Spec

The whole overhaul rides **one keystone** that all five domain plans independently diagnosed: the `power.cast` cue at `SimPlayer.gd:362` emits only `{power_id, name, pos, cue}`, so the presentation layer is blind to what a spell *did* and ~24 of 36 powers collapse into one red ring. Fix the cue, thread the aim point, and you simultaneously unlock targeting, spell archetypes, the cursor, the targeting preview, and tooltips. Everything else is built on that spine plus **one deterministic impulse solver** that every shove in the game writes through.

Dedupe note (load-bearing — these are ONE system each, not two):
- **SimForces (Combat plan) == ImpulsePhysics (Physics plan)** → ONE deterministic momentum solver, file `src/sim/ImpulsePhysics.gd`.
- **SimWell (Combat) == ForceField (Physics)** → ONE gravity-well/force-field module, `src/sim/ForceField.gd` (the Maw).
- **Five "cursor" features** (blood-drop reticle / Precognition Cursor / context cursor / Predator Cursor / Predation Cursor) → ONE cursor + ONE hover-target source. The `WorldFX` reticle at L75–84 is deleted **exactly once**, sequenced with PR #2.
- **Four "feeding" descriptions** (gulp-master-cancel / Seize→Pulse-Lock→Brink-Verdict / 4-phase taught ritual / lunge+heartbeat-throb) → ONE feeding act. The EKG *is* the gulp mini-game; the Verdict *is* kill/spare; the master-cancel is its in-combat use.
- **Consequence spine** (StyleLedger/Dispatch/Contract/Herald + Witness Cascade + SpawnDirector promotion) all use the same pattern: move board-changing staging into a **Sim-owned deterministic twin** driven by `Sim.draw_float`, fold its `state_hash()` into the 20-run gate; keep `DirectorService` as a cosmetic HUD mirror.

## 1. SPELLS + TARGETING — the cast contract (kills "every spell is a circle")

**THE CAST CONTRACT (keystone, Wave 1).** Enrich the cast cue to:
`{power_id, archetype, origin, target_pos, aim_dir, range, radius, arc, damage_type, impacts:[{pos,kind}], magnitude, color, discipline, clamped}`.
- Thread `target_pos` onto the **POWER InputAction**, defaulting to the already-serialized `aim_point` (already snapped + hashed in `SimPlayer.state_hash`), so casts know *where* you aimed, not just facing — and replays stay deterministic. Mouse world-pos flows through `InputMap.gd` → `InputAction` only; Sim never reads the mouse.
- **Zero-authoring archetype classifier** `src/data/SpellArchetype.gd::archetype_of(def)` (~12 lines) derives the archetype from fields powers already have. **11 archetypes**: `SELF_AURA/NOVA`, `GROUND_AOE`, `ENTITY_TARGET`, `PROJECTILE/BOLT`, `CONE`, `BEAM/CHANNEL`, `DOT_ZONE`, `SELF_BUFF`, `DEBUFF_VULN`, plus `WELL` and `TETHER` for physics. `color` from `DISCIPLINES[disc].color` (confirmed present).
- Out-of-range ground casts are **clamped in-sim with deterministic float math** (so the 20-run hash stays green) and the `clamped` flag pre-warns the cursor.

**3× per archetype:** each carries its own data contract AND its own SpellFX renderer with a 3-beat **anticipation → impact → aftermath**, so the eye learns a grammar in one fight. `SELF_BUFF` deliberately draws **no ground ring** (body-hugging aura + icon) so buffs stop reading as attacks. A 0.08 s "lock" reticle snaps at `target_pos` before resolve.

**Edge cases (all handled, never fizzle silently):** out-of-range → reticle clamps to max-range point (amber "too far"); wall between → snap to last open cell (red-X) via existing `resolve_motion` raycast; no valid `ENTITY_TARGET` under cursor → fall back to existing `_aim_target` auto-nearest; controller → derive `target_pos` from `facing*range`; cast-while-cloaked mutes the palette.

Files: EDIT `SimPlayer.gd` (~L362, build the contract + `_resolve_cast_target()`); EDIT `InputAction.gd` (+`target_pos`, +`PARRY` kind), `InputMap.gd`; EDIT `GameCatalog.gd` (+`archetype`+`color` keys per power, pure metadata); NEW `SpellArchetype.gd`. **No WorldFX edits.** Covered by a `state_hash` regression like the existing `cel_haste`/`bs_bolt` tests.

**HEMOMANCY (Wave 3).** Casting into/through your own spilled blood **transmutes the archetype**: Blood Bolt over a deep pool → ground-targeted geyser DoT; Earthshock on a soaked tile → crimson bleed-shockwave; Cauldron on a trail → fire-river; Theft through a pool siphons the ground for free vitae. Reads `world.blood_at(target_pos)` + at player, branches archetype against a depth threshold, **consumes** the medium via `siphon_blood`. Not enough depth → fires the NORMAL archetype (graceful). Standing in your own carnage raises exposure/Heat — a real Sanguine build axis.

## 2. PHYSICS — one impulse solver, everything rides it (kills "no real physics")

**PhysicsBody columns on SimEntity (Wave 1 substrate).** Add `angular_vel`+`spin_drag`, a `z_height`/`z_vel` vertical column with gravity (lobs/launches/pounces), `mass`, and a `physics_state {GROUNDED, TUMBLING, AIRBORNE, PINNED}`. Mass table: rat 0.3, ped 0.7, thug 1.0, swat 1.6, elder 2.4, vehicle 6.0. **Cosmetic-only** render `spin_angle` is separate from `facing` (AI aim), so a thrown thug tumbles without corrupting his aim. Per-axis drag reads the blood grid (slide far on a slick: 0.93; rough asphalt 0.84). `TUMBLING` suppresses the SimNPC brain for `tumble_ticks` then an 18-tick GET_UP. All 6 fields added to `state_hash()` with `snapped(x,0.001)`; **add physics cases to the 20-run fixture BEFORE wiring powers**, re-baseline in the same PR.

**THE IMPULSE SOLVER (signature keystone, `src/sim/ImpulsePhysics.gd`, Wave 1).** One deterministic broad+narrow-phase pass in `tick_combat`. Every shove — Tether, Maw ejection, dash shoulder-check, car bump, knockback, pounce, feed-yank — writes the existing `knockback_vel` channel. When two fast bodies overlap:
- **Proportional damage to BOTH:** `dmg = clamp(closing_speed*0.06, 4, 60)*mass_factor`. Never constant.
- **Conserved momentum:** `j = -(1+e)*(v_rel·n)/(1/m1+1/m2)`, restitution `e=0.35` body-body, `0.5` vs-wall (reflect+halve the blocked component); mass-weighted positional correction prevents overlap-stick.
- **Tumble:** tangential component injects `angular_vel` into both — glancing hits set up cartwheels.
- **Billiard chains:** one fling clips three in a row, each passing residual momentum; **capped 24 contacts/tick** to bound recursion; a "3 HIT" floater.
This is the explicit, permanent negation of the player's rejected anti-pattern (frozen sprite, constant speed, constant damage, straight line): speed decelerates via drag+bounce, damage reads closing speed, trajectory bends via spin+collisions.
**Determinism:** only "dirty" bodies (`|vel|+|knockback_vel|` over threshold) enter the scan (a calm city pays ~zero, per "world starts calm"); stable-sort pairs by `(id_a,id_b)`; **fixed 2 solver passes, never "until rest."** Gated by NEW `test/test_physics_determinism.gd` (8-body Tether pile-up + 6-body Maw, 20-run hash).

**THE TETHER (Wave 1 showcase verb).** Re-skins `shd_arms`. GRAB a body in range (lifts via z-offset, gains spin ∝ cursor-swing speed); SLING toward an aim-point at a speed set by swing-velocity (capped); IMPACT routes through the solver so BOTH bodies take damage, the struck body inherits recoil and bowls through a cluster, both spill blood, and a lit contact cell ignites both. Edges: into-wall = full energy to one body + crack-decal; into-water extinguishes + washes blood; a MARKED body detonates on landing (W3 combo); into the haven fizzles. **Determinism boundary (explicit):** impulse + per-step impact resolution live entirely in the sim via `knockback_vel`; only spin/tumble/squash is cosmetic. True many-body RIGID rotational collisions are **flagged, NOT built** (would need a Godot-physics body and accept determinism loss).

**THE MAW / gravity well (`src/sim/ForceField.gd`, Wave 2).** A `WELL` cast at a ground point opens a ~150-tick singularity. Inverse-distance suction (edge drifts you in, core yanks) PLUS a tangential **swirl** so bodies SPIRAL and tumble, not slide on a line. Caught bodies collide mid-suction via the solver (a crowd crushes itself, core-crush DoT scales with stack count); a **collapse NOVA** z-launches a fountain of bodies + a 40-pip blood geyser. The well **drinks** the blood grid inward (advection like the fire layer). A Tethered body fed into the well = lethal launch (the cross-product). Two wells SUM. Extends the determinism test.

**Piloted Locomotion (Wave 1, kills "constant-speed float").** Route the player through `entity.vel` instead of the instant `pos += dir*speed`. Three accel regimes: standing-start 2400 px/s² (0.09 s to base 220), airborne/mid-dash 600 (committed, no instant u-turn), turn accel scaled by `dot(vel_dir,input_dir)` so a 180° reversal hits 0.55× — a felt skid. Split friction: ground 0.86/tick, skid 0.94 for the first 7 ticks (slide a vampire's-length), blood/ice 0.97 (hydroplane through your own carnage). Sprint = a 2-stage spool 0→1.7× over 18 ticks. Carrying a body halves accel; frenzy raises accel 1.3× but adds overshoot. New accel/stride/skid fields → `state_hash` with `snapped()`, re-baseline same PR. `lean`/`cape`/`bob` are present-only (CharacterRig2D, PR #2).

**Quarantined spectacle ragdoll (Wave 3, the ONE named determinism-loss boundary).** At the instant of death a corpse MAY detach into a Godot `RigidBody2D` seeded with the sim body's exact `vel`/`angular_vel`/`z`, tumble under true contacts, then bake a blood decal — and is removed from the sim entirely so it can't touch the hash. Capped ~12 active, OFF in headless/determinism tests via a feature flag, collides only with walls + other ragdolls. Disabled → falls back to the sim-side tumble-and-fade. Warranted only for the Maw's pile-up money shot.

## 4. FEEDING — one visceral predation act (kills "blob says Gulp" + "red oval")

ONE feeding verb, four beats, fully deterministic, every beat with anticipation/impact/aftermath. New VFX in NEW `src/present/PredationFX.gd` (never WorldFX). **BEAT 1 — THE SEIZE (Wave 1).** Replaces the instant `target.ai_state = fed`. The player snaps to a grapple offset; the victim enters STRUGGLE with a deterministic resist meter (seeded by type: rabble 18, thug 42, cop 64). Sub-states GRAB(12t)→PIN(18t)→SINK(8t). A **stealth seize** from the unseen rear arc skips the struggle (rewards the stalk); a loud front grab gets the full fight. Three break edges: frenzied player auto-passes struggle but lethal-locks the verdict; a second hostile can RESCUE-interrupt; seizing a cop pings the Director. Struggle is gated by pallor — once drain `f>0.6` they're too weak to break free, so the **early grab is the only dangerous window**.

**BEAT 2 — THE PULSE-LOCK EKG (Wave 1).** The invisible metronome gulp window becomes a **sight-readable heartbeat trace** on the victim. You tap to lock the systole UPSTROKE, not a blind timer. Four authored waveforms by humour (sanguine broad 60bpm ~20t forgiving; choleric sharp 96bpm razor 8t with a deceptive pre-spike; melancholic 48bpm long-QT plateau punishing early taps; phlegmatic glacial 40bpm easy, least yield). **Five lock tiers** replace binary hit/miss: PERFECT (≤3t, full bonus + 0.12 s vitae-slowmo + heart-thud), GOOD, GRAZE, EARLY (trace "skips a beat"), TIMEOUT. A **Cadence** layer (reuse the `flow_stacks` pattern): 3 perfect → "In Rhythm" (+25% drain, AudioDirector locks a sub-bass kick to the systole); 5 → "Enthralled Pulse" (heart entrains to your tempo). One miss drops ONE rung, not to zero (reachable ceiling). **Real-time paling fused in:** drain fraction drives a deterministic 5-tier `pallor` on `tags['pallor']` the EntityRenderer reads — the EKG amplitude IS pallor; they sag together. So the rhythm game and the visible draining-of-life are the same read.

**BEAT 3 — THE BRINK & VERDICT (Wave 1).** Three drain stages: SATED (>40%) → THE TREMBLE (40–15%, EKG sags, soft warning) → THE BRINK (<15%, max 90t crush window: EKG flips to agonal gasps, the screen desaturates toward arterial red via **chiaroscuro** not flat vignette per the legibility memory, a Cinzel RELEASE/HOLD diptych blooms, time dilates ~0.3×). The held input becomes a **Verdict with no menu**: RELEASE = wrench-out SPARE (survivor staggers off permanently corpse-white + `traumatized`, +humanity, no kill-heat, a "Mercy" style tag, a body that remembers you). HOLD past the window = THE CRUSH, one final do-or-die Pulse-Lock: PERFECT = Clean Kill (max vitae, quiet, intact carryable corpse); FUMBLE = Botched Kill (victim SCREAMS, +50% witness radius, arterial spray). A hidden **Restraint** meta on SimMeta makes sparing build a permanent wider Brink window + a "gentle predator" rumor, so a spare is never strictly worse.

**THE GULP-AS-MASTER-CANCEL (combat context of the same verb, Wave 1).** Pressing FEED at the exact frame an enemy's guard breaks (stun/root/parried/lifted/<36% HP) is a 5-tick just-frame that cancels ANY recovery, refunds dash, grants 12 i-frames, banks +20 FLOW, and detonates the predation beat. Three outcomes: PERFECT (full payload + 2× first-gulp vitae), LATE (normal feed, the masher still works), WHIFF (a punishable 14f lunge-recover). Mid-combo variants: Lunging Bite (out of melee_light), Pinning Bite (out of melee_heavy recovery), Aerial Drain (biting a launcher-lifted enemy, rains blood that re-arms Hemomancy).

**BEAT 4 — REAL BLOOD (Wave 1).** Each lock emits a small deterministic spill at the neck cell that POOLS; the finisher arterial-SPRAYS a vectored fan; chin/mouth dribble lingers ~20 s then dries; a seize handprint/drag smear stamps where you held or hauled. All persistent blood lives in the existing `SimWorld.blood` grid (deterministic, already lit) — the red-oval grievance is solved by making the intimate act itself bleed, not by a death decal.

**THE VINTAGE ECONOMY (Wave 2).** `_apply_resonance` becomes a 3-tier escalating, stacking, dissonance-managed buff keyed off final Cadence (sloppy = tier-1; In Rhythm = tier-2 +50%; Enthralled = tier-3 + a unique rider: sanguine overheal-cap, choleric guaranteed staggering crit, melancholic next-spell-free, phlegmatic one-hit shield).
